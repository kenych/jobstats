Jobstats
========

most demanded devops skills stats, DIY approach.

Usage
-----

First:

``` jobstats.sh 'https://www.jobserve.com/gb/en/JobListing.aspx?shid=B141368A285A3CF3D6' ```

then once data is cached:

```./jobstats.sh -c  ```

to sort by numbers:

```./jobstats.sh -c -2 ```

will get you output like:

